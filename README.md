## معرفی (Introduction)

**حسابداری دانیال** یک نرم‌افزار جامع حسابداری، انبارداری و مدیریت فروش است که برای **همه نوع کسب‌وکار** طراحی شده — از فروشگاه‌های اینترنتی تا فروشگاه‌های فیزیکی، از استارتاپ‌ها تا کسب‌وکارهای بزرگ.

> **حسابداری هوشمند، فروش حرفه‌ای**
> **Smart Accounting, Professional Sales**

## چرا حسابداری دانیال؟

| ویژگی | توضیح |

## معماری و قابلیت‌ها

### حسابداری (Accounting)
- دفتر حسابها با ساختار درختی
- ثبت اسناد حسابداری (روزنامه)
- تراز آزمایشی
- صورت سود و زیان
- ترازنامه
- گردش وجوه نقد
- گزارش سنی مطالبات مشتریان
- مدیریت دوره‌های مالی
- همگام‌سازی خودکار فروش و خرید با دفتر کل
- تاریخچه تغییرات قیمت خرید و فروش

### انبارداری (Inventory)
- مدیریت محصولات با بارکد و QR
- چاپ لیبل بارکد و QR
- گزارش موجودی و ارزش ریالی انبار
- کالاهای کندفروش
- تاریخچه تغییرات (Audit Log)
- چاپ A4 و خروجی اکسل

### فروش (Sales)
- صندوق فروش (POS) با اسکنر بارکد
- فاکتور الکترونیکی و چاپی
- انواع فروش: حضوری (پیشخوان) و آنلاین
- پرداخت نقدی و اعتباری
- مرجوعی کالا
- تاریخچه فروش با جستجو و فیلتر بر اساس نوع فروش

### مشتریان و تأمین‌کنندگان
- مدیریت مشتریان با حساب دفتری (حقیقی/حقوقی)
- مدیریت تأمین‌کنندگان با حساب دفتری
- پیگیری بدهی و طلب
- پرداخت و دریافت با ثبت خودکار حسابداری

### امنیت و پشتیبانی
- رمزنگاری AES-256
- امضای دیجیتال فایل‌ها
- پشتیبان‌گیری خودکار با قابلیت حذف
- بازیابی با اعتبارسنجی نسخه
- خروجی چند فرمت: SQLite (.db) / JSON (.json)
- انتخاب محدوده داده: همه / فقط ساختار / انتخابی
- مهاجرت خودکار دیتابیس هنگام ارتقا نسخه
- لاگ عملیات (Audit Trail)

### صادرات و واردات هوشمند
- انتخاب بخش‌های دلخواه برای صادرات/واردات
- تشخیص خودکار وابستگی داده‌ها
- ۳ مرحله خروجی: مکان ذخیره → فرمت → محدوده داده
- پیش‌نمایش قبل از واردات با نمایش نسخه و تعداد جداول
- مهاجرت خودکار بین نسخه‌ها
- پیش‌نمایش قبل از واردات
- اعتبارسنجی و بررسی یکپارچگی

## تکنولوژی‌ها

| لایه | تکنولوژی |
|------|----------|
| **فرانت‌اند** | React 18 + TypeScript 5 + Tailwind CSS |
| **بک‌اند** | Electron 33 + Node.js |
| **پایگاه داده** | SQLite (better-sqlite3) |
| **وضعیت** | Zustand 5 |
| **بیلد** | Vite 6 |
| **تقویم** | jalaali-js (Shamsi/Jalali) |
| **فونت** | Vazirmatn (local woff2) |

## نصب و راه‌اندازی

```bash
# کلون کردن مخزن
git clone https://github.com/danialchoopan/DukanElectronJS.git
cd DukanElectronJS

# نصب وابستگی‌ها
npm install

# اجرای در حالت توسعه
npm start

# ساخت نسخه اجرایی
npx electron-builder --win
```

## ساختار پروژه

```
src/
├── main/                    # Electron main process
│   ├── database/            # SQLite schema, repos, seed, backup
│   │   ├── connection.ts    # 20+ tables, all columns in CREATE TABLE
│   │   ├── repositories/    # CRUD for all modules
│   │   ├── backup.ts        # Backup/restore/verify
│   │   └── smartExport.ts   # Selective import/export
│   ├── ipc/                 # IPC handlers
│   └── index.ts             # App entry point
├── renderer/                # React UI
│   └── src/
│       ├── components/
│       │   ├── ui/          # Button, Dialog, Card, etc.
│       │   ├── layout/      # Sidebar, Header, etc.
│       │   ├── print/       # Print dialogs and preview
│       │   └── business/    # Domain-specific (scanner, date picker)
│       ├── views/           # Page components
│       │   ├── accounting/  # Journal, Balance, Income, etc.
│       │   ├── settings/    # Customization, Backup, etc.
│       │   └── ...
│       ├── store/           # Zustand stores
│       ├── hooks/           # Custom hooks
│       ├── utils/           # Utilities (jalali, print, QR, etc.)
│       ├── i18n/            # Translations (FA/EN)
│       └── App.tsx          # Root component
├── preload/                 # Electron preload scripts
└── types/                   # TypeScript types
```
