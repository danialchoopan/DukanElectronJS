API Reference (IPC + Preload)

All communication between renderer (UI) and main process (Node.js) goes through IPC. The preload script exposes window.api with typed namespaces.

Pattern

// handlers.ts (main process)
ipcMain.handle('products:getAll', () => products.getAllProducts())

// preload/index.ts
products: { getAll: () => ipcRenderer.invoke('products:getAll') }

// component (renderer)
const r = await window.api.products.getAll()
if (r.success && r.data) setProducts(r.data)

All methods return Promise<IPCResponse<T>> where IPCResponse = { success: boolean; data?: T; error?: string }.

Products API

products.getAll()—Product[]All active products sorted by title
products.getById(id)numberProductSingle product by ID
products.getByBarcode(barcode)stringProductLookup by barcode
products.search(query)stringProduct[]LIKE search on title, barcode, category, subcategory
products.create(data)ProductInputProductCreate product (auto-barcode if blank)
products.update(id, data)id, PartialProductUpdate product fields, logs price changes
products.delete(id)numberbooleanSoft delete (isActive=0)
products.updateStock(id, qty)id, changebooleanAdditive stock change
products.getLowStock()—Product[]Products where stock <= minStock
products.getSellable()—Product[]Products with isSellable=1 (for POS)
products.bulkImport(products)arrayresultsUpsert by barcode

Sales API

MethodParamsReturnsDescription
sales.create(data)SaleInputSaleAtomic: insert sale + items + stock + journal + ledger
sales.getById(id)numberSaleSale with items and user/customer names
sales.getByDateRange(start, end)datesSale[]Sales in date range (by saleDate)
sales.getDailySummary(date)stringsummaryTotal, cash, card, ledger for a day

SaleInput includes: saleDate? for backdated invoices, affectsInventory? to skip stock changes.

Customer / Supplier API

MethodDescription

customers.charge(id, amount, desc)Add debt/credit + ledger entry
customers.pay(id, amount, desc)Record payment + ledger entry
suppliers.getAll()All active suppliers
purchases.create(data)Atomic: insert purchase + stock + supplier balance + journal
suppliers.pay(id, amount)Record payment to supplier

Backup API

MethodDescription
backup.create()Quick SQLite backup to backups/ folder
backup.saveAs()Save backup to user-chosen path
backup.createWithOptions(opts)Enhanced export: format (sqlite/json), scope (all/structure/selective), tables
backup.restore(path)Restore from file (auto pre-backup)
backup.getSelectableTables()List tables with row counts for selective export
backup.list()All backups with meta (name, size, timestamp, hash)
backup.integrity(path?)PRAGMA integrity_check + foreign_key_check
